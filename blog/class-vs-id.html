css selectors: class or id?

what are they?

they are two types of CSS selectors which are used to point to certain specific HTML elements so you can apply some styling to them

how and when to use id

id selector is used to select and target a single HTML element with a unique id attribute. It's unique so it can only be applied to this specific HTML element.

Like your IRD tax number - it belongs and is unique to you. Noone else can use it to identify themselves and so this HTML element gets its own unique identifier in the form of a an id selector.

so when to use id and when to use class ?

best practice is to use id for single elements that appear on the page only once like a header or footer. Each element can have only one ID and each page can have only one element with that ID.

use class for single or multiple elements that appear once or more like paragraphs.

keep in mind as well that in css world, id selectors have a higher specificity than attribute selectors becuse it's well, more specific :)
